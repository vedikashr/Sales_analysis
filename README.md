# 📊 Sales Data Analysis Project

Exploratory Data Analysis of sales dataset using Python
## 🔍 Objective

Analyze sales data to uncover trends, regional performance, and category insights.

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib

## 📈 Analysis Performed

* Monthly sales trends
* Sales by region
* Sales by category
* Region vs Category analysis

## 💡 Key Insights

* Sales vary across months indicating seasonal trends
* Certain regions contribute more revenue than others
* A specific category dominates total sales
* Regional performance depends on category demand
